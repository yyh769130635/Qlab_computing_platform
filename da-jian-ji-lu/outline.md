# Outline

大数据有许多工具，适用不同场景，实验室服务器上已经搭建可用的有以下一些：

* Hadoop
* Spark
* hive && mysql
* zeppelin
* jupyter notebook
* OpenPai
* docker
* Kubernetes

![](../.gitbook/assets/image%20%2853%29.png)

同学们可以根据自己的需求，使用不同的工具。

整个文件夹记录了常用大数据工具搭建的过程，以及其常见的使用方式，欢迎补充！ 😇
